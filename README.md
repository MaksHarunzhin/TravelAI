# TravelAI
Учебный проект предназначенный для персонализированных рекомендаций мест отдыха (посредством диалоговой системы: чат-бот или Web-интерфейс) и развлечений на основе предпочтений пользователей и их опыта. В рамках проекта необходимо разработать архитектуру системы с поддержкой пользовательского контента (UGC), рейтинговой системой и интеграцией LLM для естественного диалога и суммаризации отзывов. Организовать автоматический парсинг информации из внешних источников (Telegram-каналы/веб-порталы), реализовать ролевую модель доступа и систему рекомендаций на основе категорий и рейтингов. Предусмотреть механизмы модерации контента, кэширование популярных запросов к LLM, эффективную индексацию для быстрого поиска по категориям и фильтрации мест.
